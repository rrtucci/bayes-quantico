\chapter{Orthogonal Groups}
\label{ch-ortho-groups}
This chapter is based on Cvitanovic's Birdtracks book Ref. \cite{birdtracks-book}.

$SO(n)$
is the group of
matrices
under whose transformation, the following  symmetric quadratic form,
is invariant

\beq
h(p, q)=
g_{\mu \nu}p^\mu q^\nu
\eeq
Here 

\beq
g_{\mu\nu}=g_{\nu\mu},\quad
g^{\mu\nu}=g^{\nu\mu}
\eeq

\beq
g_{\mu\s} g^{\s \nu}=\delta_\mu^\nu
\eeq

\beq
g_{\mu\nu}=
\xymatrix
{&g^\dagger\ar[l]\ar[r]&
},
\quad
g^{\mu\nu}=
\xymatrix
{\ar[r]&g&\ar[l]
}
\eeq

\beq
\myboxed{
g_{\mu\s} g^{\s \nu}=\delta_\mu^\nu}
\quad
\xymatrix{
&g^\dagger
\ar[l]\ar[r]
&g
&\ar[l]
}=
\xymatrix{&\ar[l]}
\eeq
Use only lower indices in all
matrices except the metric. Use
the metric with
upper indices only.

\beq
g_{\mu\nu}=
\xymatrix
{&g^\dagger\ar@{-}[l]\ar@{-}[r]&
},
\quad
g^{\mu\nu}=
\xymatrix
{\ar@{-}[r]&g&\ar@{-}[l]
}
\eeq

\beq
\delta_\mu^\nu
=\xymatrix{
\mu& \ar@{-}[l]|\bullet
\nu
}
=\xymatrix{
\mu& \ar@{-}[l]
\nu
}
\eeq

\beq
\myboxed{
g_{\mu\s} g^{\s \nu}=\delta_\mu^\nu}
\quad
\xymatrix{
&g^\dagger
\ar@{-}[l]\ar@{-}[r]
&g
&\ar@{-}[l]
}=
\xymatrix{&\ar@{-}[l]}
\eeq
