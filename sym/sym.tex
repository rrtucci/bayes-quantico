represented by the following $2!=2$ birdtracks\footnote{Note
that the set of values that
$a_i$
and $b_i$

as long as, for some set $V$,

\indi\indices{
_{a_1,a_2}

&\bullet\ar@[green][l]
&{b_1}\ar[l]
\\
{a_2}
&\bullet \ar[l]
&{b_2}\ar[l]}
\ecen
\eeq

$a^{:p}$ is

\beq
\eps^{123\ldots p}=
\eps_{123\ldots p}=1
\eeq

\beq
